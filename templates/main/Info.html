    Send auto-report text message

    응급상황이 감지되었을 때 일정시간 동안 응답이 없으면 자동으로 신고 접수가 됩니다.

    GPS

    GPS기반 현재 위치를 주변 경찰서와 소방서에 알려 최대한 빠르게 도와드립니다.

    Accurate Analysis

    100만개의 데이터를 분석하고 딥러닝을 통해 소음과 위급상황 소리를 분류해서 정확도를 높이려고 했습니다.

    HOW TO USE

    민감도/카운트 설정

    - 주변 소리에 대해 민감도를 설정할 수 있습니다.
    - 설정한 카운트에 따라 위급상황시 신고 접수 빠르기를 설정할 수 있습니다.
    설정 > 민감도 / 카운트 설정

    스위치 ON/OFF

    - 사용자가 서비스의 기능을 키고 싶을 때 => ON​
    - 사용자가 서비스의 기능을 끄고 싶을 때 => OFF
    ON상태일 때 화면에서 벗어나도 서비스 기능이 작동 합니다.

    오감지될 경우

    위급상황 소리를 감지했을 때 예/아니오 팝업이 뜹니다. 위급상황이 맞다면 '예' 아니면 '아니오'를 눌러서 오신고가 되지 않도록 유의해주세요.
    버튼을 사용자가 설정한 카운트에 따라 일정시간 누르지 않으면 자동신고가 접수 됩니다.
